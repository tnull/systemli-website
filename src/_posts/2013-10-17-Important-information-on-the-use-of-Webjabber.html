We use the software Jappix. for the Webjabber. This software normally stores various data like conversations in the local part of your browser. This feature is not necessary but actually quite dangerous in case you use a computer that is not encrypted. This is why we have altered this feature of the software to that effect that no data is stored in your browser. For the sake of transparency we provide you with the opportunity to check for yourself if there is any data of the Webjabber stored in your browser and to delete them if necessary.

This information exclusively refers to the use of the Webjabber and does not concern the use of a native client like Pidgin, Psi or Adium.
